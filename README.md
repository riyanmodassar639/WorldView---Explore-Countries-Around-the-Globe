# 🌍 WorldView - Explore Countries Around the Globe

## Overview

WorldView is a modern and responsive web application that allows users to explore countries from around the world. Built with HTML, CSS, and JavaScript, it fetches real-time country data from the REST Countries API and presents essential information such as flags, capitals, populations, and regions in a clean and user-friendly interface.

---

## Features

* Modern and responsive UI
* Browse countries from around the world
* Search countries by name
* Display country flags
* View country population
* View capital city
* Display continent/region
* Real-time data using REST Countries API
* Fast and interactive user experience
* Mobile-friendly design

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* REST Countries API

Navigate to the project

```bash

```

Run the project
